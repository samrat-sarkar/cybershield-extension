CyberShield - Phishing Detector

What is CyberShield?

CyberShield is an advanced Chrome extension designed to detect phishing websites and protect users from online threats.

How does it work?

CyberShield analyzes website URLs, checks IP reputation, and scans for malicious activity using AbuseIPDB and VirusTotal. It provides real-time alerts and risk assessments to help users stay safe online.
